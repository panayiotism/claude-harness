# Release v3.7.0 - Command Consolidation

## What's Changed

### Features
- **Command Consolidation** (#22) @panayiotism
  - Reduced from 13 to 6 commands for streamlined UX
  - `/do` now handles both features AND fixes via `--fix` flag
  - Renamed `/merge-all` to `/merge`
  - Auto-reflect always enabled at checkpoint

### New 6-Command Structure
| Command | Purpose |
|---------|---------|
| `/setup` | Initialize harness (one-time) |
| `/start` | Compile context, show status |
| `/do` | Unified workflow (features + fixes) |
| `/checkpoint` | Manual commit + push + PR |
| `/orchestrate` | Multi-agent team (advanced) |
| `/merge` | Merge PRs, auto-version, release |
