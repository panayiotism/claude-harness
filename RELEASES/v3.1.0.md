# Release v3.1.0

## What's Changed

### Features
- **Bug Fix Command** (`/claude-harness:fix`) - Create bug fixes linked to original features (#10)
  - Fixes share memory context with original feature
  - GitHub issue integration (linked to original feature issue)
  - Uses same agentic loop as `/implement` with `fix:` commit prefix
  - PATCH version bumps in `/merge-all`

### Memory Architecture (v3.0.0)
- 4-layer memory: Working, Episodic, Semantic, Procedural
- Failure prevention system (learns from mistakes)
- Test-driven features (generate tests before implementation)
- Impact analysis for breaking changes
- Context compilation each session

### New Commands
- `/claude-harness:fix` - Create linked bug fixes
- `/claude-harness:plan-feature` - Plan feature implementation before coding
- `/claude-harness:generate-tests` - Generate test cases before implementation
- `/claude-harness:check-approach` - Check if approach matches past failures
